# Threaded Sobel Edge Detection
this project is a simple sobel filter, with optimizations to run on the raspberry pi 3 B+.

### To Build:

#### Dependencies:
make
gcc
pkg-config
opencv4
pthread compatability

#### Building:
`make` to build on your own system 
executable will be named runMe

### Running Sobel
> ./runme ./path/to/.avi
